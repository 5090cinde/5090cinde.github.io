Verslag

Project

Rc auto met Gesture control met bijbehorende website

Doelen:

- Binnen de tijd ons project af hebben.
- Product werkt naar wens.
- Aantrekkelijk product.

Milestones:

- Welke hardware werkt goed met microbit.
- Onderzoek 2 microbits met elkaar combineren/praten.
- Onderzoek hoe je een motor kunt aansturen met microbit.
- Onderzoek hoe je software kun beschrijven van microbit.


Ambities:

- Weten hoe we een rc auto met gesture control kunnen maken.
- Weten hoe een goed project tot stand komt.
- Weten hoe een goed project moet verlopen.

Wat hebben we geleerd van dit project?

Roxanne: Ik heb van dit project geleerd dat een goede planning en een logboek cruciaal
zijn. Zo kunnen we alles makkelijker controleren en bekijken waar het eventueel fout is
gegaan. Verder heb ik ook geleerd hoe je leiding kan nemen in een groep en de eerste stap
kan zetten. Ook de communicatie binnen een project is belangrijk en ook hier heb ik geleerd
hoe ik de eerste stap kan zetten. Dan is een website maken ook een groot leerpunt geweest
in het proces. Ik heb geleerd hoe ik een website kan maken dat op een hoger niveau voor mij.

Cinde: Van dit project heb ik geleerd hoe ik leiding kan nemen in een groep, hoe ik op hoger
niveau een website kan maken, hoe ik een overzichtelijke planning kan maken en wat
een goede planning voor een impact heeft op het project. ten slotte heb ik geleerd hoe belangrijk
overleggen is in een project en hoe je dat het best kunt doen.

Bart V.: Dit project heeft mij vooral geleerd dat informatica breder en moeilijker is dan achter
een computer zitten. Ik vond het extreem moeilijk, wat voor mij soms demotiverend werkte.
Ik vond het wel erg interessant, maar ben erachter gekomen dat ik de informatica aan
anderen overlaat. Ik heb ook geleerd dat samenwerking erg belangrijk is, aangezien ik graag
op mezelf ben. Een plan is ook erg belangrijk en je moet je hieraan dan ook goed houden
om het project binnen goede banen te houden.

Michael: Ik heb hiervan geleerd dat we goed moeten samenwerken en duidelijke afspraken
moeten maken wie wat doet. In het begin hadden we daar moeite mee omdat we elkaar niet echt
kende. We wisten ook niet wat iedereen kon. Later ging dat beter en heeft iedereen een
steentje bijgedragen aan dit project.

Hugo: In dit project heb ik geleerd hoe belangrijk plannen is. Gelukkig zijn we begonnen
met het maken van een plan van aanpak en een taakverdeling. Hoewel de taakverdeling is
veranderd in de loop van tijd, was het belangrijk dat we wisten wie wat deed. Op deze
manier werd ieders kwaliteiten optimaal benut.

Cedric: Informatica is niet mijn ding en ik ga hier in de toekomst niks mee doen.

Bart B: Wat ik geleerd heb van dit project is niet alleen dat samenwerking heel belangrijk is.
Maar dat programmeren ook heel leuk is. In het begin hadden wij wat problemen maar een paar lessen in het project begon het steeds meer als een
machine te draaien. Ook was het
een hele uitdaging omdat wij elkaar voor het project niet echt goed kende wat zorgde voor
een paar communicatie problemen. Maar eind goed al goed, ik heb veel geleerd van dit
project en het heel erg naar mijn zin gehad.
